## Changelog
Most of the features shown in the video are listed below:

<table>
  <thead>
  <tr>
    <th style={{ width: '180px' }}>Change</th>
    <th>Description</th>
  </tr>
  </thead>
  <tbody>
    <tr>
      <td>Delete Node</td>
      <td>Allows the user to delete a selected Node from the canvas.</td>
    </tr>
    <tr>
      <td>Popup Menu</td>
      <td>Displays a context menu when the user clicks on a Node.</td>
    </tr>
    <tr>
      <td>Reset New Node</td>
      <td>Returns the new Node dot to its original position after the user drops a new Node onto the canvas.</td>
    </tr>
    <tr>
      <td>Drag New Node</td>
      <td>Enables the user to add a new Node by dragging it.</td>
    </tr>
    <tr>
      <td>Click New Node</td>
      <td>Allows the user to create a new Node close to its parent by simply clicking on a new Node.</td>
    </tr>
    <tr>
      <td>Bezier Curve</td>
      <td>Enhances the appearance of lines connecting Nodes by using the Bezier curve algorithm.</td>
    </tr>
    <tr>
      <td>Curved Lines</td>
      <td>Creates a curved line between two Nodes based on the position of the connecting Nodes.</td>
    </tr>
    <tr>
      <td>Connect Nodes</td>
      <td>Allows the user to connect Nodes either vertically or horizontally, maintaining a consistent orientation.</td>
    </tr>
    <tr>
      <td>Preview New Node</td>
      <td>Displays the final position of a newly created Node before the user drops it onto the canvas.</td>
    </tr>
    <tr>
      <td>Update Connection</td>
      <td>Automatically updates the connection points of lines when Nodes are moved around the canvas.</td>
    </tr>
    <tr>
      <td>Line Label</td>
      <td>Enables the user to add a label to the line connecting two Nodes, providing additional flow context.</td>
    </tr>
    <tr>
      <td>Snap on Drop</td>
      <td>Enhances the drag and drop visual effect by allowing the user to freely drag Nodes without snap effect, but snaps the Node into place once the user drops it onto the canvas.</td>
    </tr>
    <tr>
      <td>Snap to Grid</td>
      <td>Enables Nodes to snap to a grid pattern to ensure they are aligned properly.</td>
    </tr>
    <tr>
      <td>Grid Pattern</td>
      <td>Adds a dotted grid pattern to the canvas to aid in aligning Nodes and lines.</td>
    </tr>
    <tr>
      <td>Move Nodes</td>
      <td>Enables the user to create and move Nodes with connection lines, using drag and drop interactions.</td>
    </tr>
    <tr>
      <td>WIP - Select Nodes</td>
      <td>Enables the user to select one or multiple Nodes. This feature is currently a work in progress.</td>
    </tr>
    <tr>
      <td>WIP - Input Text</td>
      <td>Enables the user to input text into a Node. This feature is currently a work in progress.</td>
    </tr>
  </tbody>
</table>
